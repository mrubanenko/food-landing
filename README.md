# Food Landing

Responsive landing page about food, built with **HTML** and **SCSS**.

## Features

- Fully responsive layout for mobile, tablet, and desktop  
- Modern CSS using **SCSS**, with **variables, mixins, and partials**  
- **BEM methodology** for semantic and maintainable class naming  
- Clean and semantic HTML structure  
- Adaptive design with flexible grid and media queries  
- Easy to customize and extend
